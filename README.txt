GenDunExp (GDE)
A java program in which the user creates a party of adventurers to explore a randomly generated dungeon.
Created and tested using Eclipse IDE

To run:
	download GDE.jar, ensure that the machine running the program has java installed
	open a cmd prompt, navigate to the directory containing the jar file,
	run via java -jar GDE.jar

	User interaction is guided via prompts within the game
